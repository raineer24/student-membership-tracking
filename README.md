# 🎓 Student Monthly Membership Tracking System

A backend API for managing student memberships, payments, and overdue tracking. Built with **Node.js**, **Express**, **Prisma**, and **PostgreSQL** (with support for **Supabase** in production).

---

## 🧩 Features

- ✅ **JWT Authentication**
- ✅ **Role-Based Access Control**: Admin, Student
- ✅ **Student Management** (CRUD)
- ✅ **Membership Tracking**
- ✅ **Payment History**
- ✅ **Overdue Student Detection**
- ✅ **Supabase PostgreSQL Support** for Vercel deployment

---

## 🛠️ Tech Stack

| Layer | Technology |
|------|------------|
| Backend | Node.js + Express |
| Database | PostgreSQL (Local or Supabase) |
| ORM | Prisma |
| Auth | JWT |
| Scheduler | node-cron (optional) |
